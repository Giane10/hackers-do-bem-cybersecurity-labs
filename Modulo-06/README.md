# Módulo 6: Proteção Web e Desenvolvimento Seguro 🛡️

Este repositório contém os laboratórios práticos realizados durante o Módulo 6 do programa **Hackers do Bem**, focados em segurança de aplicações web, automação de infraestrutura e Ciclo de Vida de Desenvolvimento Seguro (SDLC).

## 🚀 Competências Desenvolvidas

### 🐍 Automação e Scripting (Python & PowerShell)
* **Reuso de Código:** Implementação de scripts Python utilizando a biblioteca **Pillow (PIL)** para manipulação automatizada de imagens, focando em componentes validados e eficientes.
* **Auditoria de Sistemas:** Desenvolvimento de automação para coleta de metadados de hardware e Kernel no Kali Linux.
* **Administração Windows Server:** Criação de scripts em PowerShell para extração automatizada de logs de eventos (Event IDs 6005/6006) e inventário de hardware via CIM Instance.

### ☕ Ambiente de Desenvolvimento Seguro
* **Setup de Desenvolvimento:** Configuração do **OpenJDK (JDK)** e do IDE **Apache NetBeans** em ambiente Linux para desenvolvimento profissional em Java.
* **Análise de Vulnerabilidades em Bibliotecas:** Pesquisa técnica sobre o **CVE-2017-12099 (Integer Overflow)** utilizando bases de dados mundiais (NVD e CVE Details).

### 🌐 Segurança Ofensiva e Defensiva Web (OWASP Top 10)
* **SQL Injection (SQLi):** Exploração avançada com **SQLmap** para extração de dados sensíveis de bancos de dados MySQL e bypass de autenticação.
* **Clickjacking (UI Redressing):** Prova de conceito utilizando camadas `<iframe>` invisíveis e análise de cabeçalhos de segurança como **X-Frame-Options** e **CSP**.
* **Análise de Protocolo:** Manipulação de requisições HTTP e utilização de **Percent Encoding** (URL Encoding) para transferência de parâmetros especiais.
* **Integração de APIs:** Automação de encurtamento de URLs e geração de **QR Codes** via terminal para portabilidade de dados.

### 🛡️ Ferramentas de Proteção
* **Antimalware:** Operação do **ClamAV** e utilitário `sigtool` para análise de assinaturas de malware e detecção de ameaças conhecidas.

## 🛠️ Tecnologias Utilizadas
* **Linguagens:** Python 3, PowerShell, Java, SQL, HTML/CSS.
* **Sistemas:** Kali Linux, Windows Server 2022.
* **Ferramentas:** SQLmap, ClamAV, Apache NetBeans, Curl, Qrencode.
* **Frameworks de Segurança:** OWASP, NVD, CVE Details.
