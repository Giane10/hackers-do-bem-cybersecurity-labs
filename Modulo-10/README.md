# Módulo 10: Segurança de Sistemas e Aplicações 🖥️📦

Este repositório documenta as implementações práticas de isolamento de ambientes, gerenciamento de containers e endurecimento (hardening) de sistemas realizadas durante o Módulo 10 do programa **Hackers do Bem**. O foco foi garantir a segurança no nível da aplicação e do sistema operacional.

## 🚀 Competências Desenvolvidas

### 🪟 Virtualização e Isolamento (Hypervisors)
* **Gestão de Máquinas Virtuais:** Configuração avançada do **Oracle VM VirtualBox**, incluindo gestão de recursos, snapshots e isolamento de rede para ambientes de teste seguros.
* **Simulação de Hardware:** Manipulação de interfaces de BIOS/UEFI para compreensão de processos de boot seguro e ordem de inicialização de dispositivos.

### 🔐 Criptografia de Disco (Full Disk Encryption)
* **Proteção de Dados em Repouso:** Implementação de partições criptografadas utilizando o padrão **LUKS (Linux Unified Key Setup)** no Kali Linux.
* **Gestão de Volumes:** Criação de volumes lógicos seguros, montagem de sistemas de arquivos criptografados e práticas de segurança para chaves de acesso.

### 🐳 Virtualização em Nível de SO (Docker)
* **Gerenciamento de Containers:** Operação prática de containers **Docker**, incluindo a criação, execução e interrupção de instâncias isoladas (MeuContainer).
* **Segurança de Aplicações:** Entendimento das vantagens de segurança e portabilidade ao migrar infraestruturas físicas para arquiteturas de containers stateless.

### 🛠️ Tecnologias Utilizadas
* **Virtualização:** VirtualBox, DebianNetinst.
* **Containers:** Docker (CLI).
* **Criptografia:** LUKS, `cryptsetup`, Gestor de arquivos Thunar.
* **Sistemas:** Kali Linux, Windows.
