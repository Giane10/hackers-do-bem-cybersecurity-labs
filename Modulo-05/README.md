# Módulo 5: Gerenciamento de Identidades e Contas 🔑

Neste módulo do programa **Hackers do Bem**, foquei na administração avançada de sistemas, implementando controles de acesso, políticas de segurança e gerenciamento de diretórios em ambientes corporativos e Linux.

## 🚀 Laboratórios Práticos Realizados

### 🪟 Windows Server 2022 (Active Directory)
* **Controlador de Domínio:** Instalação e configuração do Active Directory Domain Services (AD DS).
* **Estrutura Organizacional:** Criação de Unidades Organizacionais (OU), Grupos e Usuários para gestão hierárquica.
* **Políticas de Grupo (GPO):** Implementação de políticas de complexidade de senha, histórico e tamanho mínimo através do Group Policy Management.
* **Gestão de Acesso:** Configuração de permissões NTFS e compartilhamento de rede para usuários do domínio.
* **Interoperabilidade:** Integração de estações de trabalho Windows (Client) ao domínio corporativo.

### 🐧 Kali Linux (Administração e Segurança)
* **Controle de Acesso Discricionário (DAC):** Gestão de permissões de arquivos e diretórios utilizando comandos `chmod` e `ls -l`.
* **Controle de Acesso Mandatário (MAC):** Implementação e auditoria de perfis de segurança com **AppArmor**.
* **Políticas de Senha:** Configuração de rotação de senhas, prazos de validade e avisos de expiração editando o arquivo `/etc/login.defs`.

## 🛠️ Tecnologias e Ferramentas
* **Sistemas Operacionais:** Windows Server 2022, Windows 10/11, Kali Linux.
* **Serviços de Diretório:** Active Directory, GPO, Unidades Organizacionais (OU).
* **Segurança Linux:** AppArmor (MAC), Permissões rwx (DAC), login.defs.
* **Protocolos e Ferramentas:** RDP, SMB, ICMP, DNS.
