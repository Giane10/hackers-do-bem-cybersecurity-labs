# Módulo 4: Gestão de Identidades e Controles de Acesso (IAM) 🔐

Neste módulo do programa **Hackers do Bem**, explorei profundamente os processos de Identificação, Autenticação, Autorização e Accounting (IAAA), aplicando conceitos de segurança em ambientes Windows, Linux e redes.

## 🚀 Laboratórios Práticos Realizados

### 🐧 Linux (Kali Linux)
* **Quebra de Senhas (Offline):** Utilização do **John the Ripper** para realizar ataques de dicionário contra hashes armazenados no `/etc/shadow`.
* **Servidor RADIUS:** Implementação e teste de autenticação centralizada (AAA) utilizando **FreeRADIUS** e o comando `radtest`.
* **Autenticação Multifator (MFA):** Configuração de **TOTP** com **Google Authenticator** e gerenciamento de tokens via **OTPClient** e plugins no Firefox.
* **Segurança do Sistema:** Auditoria de presença de hardware de segurança (**TPM**) e exploração do **SELinux** (Security-Enhanced Linux) para controle de acesso obrigatório (MAC).
* **Cofres de Senhas:** Gestão de credenciais criptografadas (AES-256) com **KeePassXC**.

### 🪟 Windows Server 2022
* **Credential Manager:** Gestão, backup e auditoria de credenciais de rede e web no ambiente Windows.
* **Protocolos de Autenticação:** Estudo prático de **Kerberos** e **NTLM** no Active Directory.

## 🛠️ Tecnologias e Ferramentas
* **Segurança:** John the Ripper, FreeRADIUS, Google Authenticator, KeePassXC, SELinux.
* **Protocolos:** RADIUS, PAP, CHAP, MS-CHAPv2, SSH, LDAP, TOTP.
* **Sistemas:** Kali Linux, Windows Server 2022.
